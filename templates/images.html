사용 설명서


프로젝트명 : 맹모삼천지교

설명 : 경기 남부권의 학교, 학원정보를 얻어오는 API 입니다.

출력 되는 데이터 값 : JSON 형식으로 제공

sigun : 시, emd : 읍면동구, types : 구분, named: 이름, daepyo: 대표자이름, telno : 전화번호, address : 주소, lat : 위도, loat : 경도

옵션 필요값 : 이름 :named=원하는이름, 시 : sigun=원하는시, 읍면동 : emd=원하는읍면동

example : 시이름으로 정보검색 = baseurl/resource_location/수원시

1. POST :

학원/학교 정보 등록 : baseurl/resource_creation/옵션

2. GET :

학원,학교 이름을 넣어 주변 5km내의 교육시설(학교,학원) 검색 : baseurl/resource_distance/이름

이름으로 학교, 학원 정보 검색 : baseurl/resource/이름

시 이름으로 학교, 학원 정보 검색 : baseurl/resource_location/시이름

읍면동으로 학교, 학원 정보 검색(학교 미지원) : baseurl/resource_emd/읍면동구이름

3. UPDATE :

이름으로 학교/학원 검색 후 데이터 수정 : baseurl/resource/옵션

4. DELETE :

이름으로 학교/학원 검색 후 데이터 삭제 : baseurl/resource/이름

저희가 가진 모든 데이터 출력 : baseurl/resource/resource_check
